搭建本站

本站主要是采用了Hexo官方主题中的webstack，由于webstack仅仅是一个网站导航的样式，没有连接文章，但是博主比较喜欢该主题
的布局风格，所以自己对该主题改造成了适合作为博文站点的主题hexo-theme-webstack。 下面就关于本站的
这个主题使用进行说明.

采用 hexo-theme-webstack主题：

修改 hexo_demo/_config.yml

重启Hexo Server

$ hexo generate
$ hexo server

再次打开本地web页面 http://localhost:4000，如图：

hexo-theme-webstack 使用说明

  • 新增或删除左侧导航目录

根据你自己的需求，修改 _config.yml中 menu的配置

  • 新增文章

修改文章模板格式 hexo_demo/scaffolds/post.md

然后，执行下面命令新建一片文章

执行后，会生成 hexo_demo/source/_posts/hello.md ，示例如下：

  • title : 就是将来文章列表中显示的文章标题
  • type ： 是文章的分类
  • description ： 文章的简介描述，将来会显示在文章列表中每篇文章的一个简单介绍

当然，你也可以直接在 hexo_demo/source/ 目录下直接新建 xxx.md的方式新增文章，而且可以设置目录，把统一类别的文章放在同一个文件夹下，hexo new 的方法不能指定文章类别.通常建议自己手动新建文章
