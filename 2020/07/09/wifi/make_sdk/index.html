安装repo

这里使用的清华大学的镜像站，repo的官方镜像站需要翻墙

apt-get install git #如果git没有安装需要先安装
curl https://mirrors.tuna.tsinghua.edu.cn/git/git-repo -o repo
chmod +x repo
mv repo /usr/bin/

设置git-repo仓库的URL为清华大学镜像站

export REPO_URL='https://mirrors.tuna.tsinghua.edu.cn/git/git-repo'

因为repo的运行过程中会尝试访问官方的git源更新自己，如果不能翻墙，那么接下来使用repo命令就会卡住

Create the 32-bit QSDK build

install depens packages

root@debian:~/wifi/qca-networking-2020-spf-11-1_qca_oem.git# sudo apt-get install gcc g++ binutils patch bzip2 flex make gettext pkg-config unzip zlib1g-dev libc6-dev subversion libncurses5-dev gawk sharutils curl libxml-parser-perl python-yaml ocaml-nox ocaml ocaml-findlib libssl-dev libfdt-dev

root@debian:~/wifi/qca-networking-2020-spf-11-1_qca_oem.git# sudo apt-get install device-tree-compiler u-boot-tools

Install the different feeds in the build framework:

root@debian:~/wifi/qca-networking-2020-spf-11-1_qca_oem.git# cd qsdk
root@debian:~/wifi/qca-networking-2020-spf-11-1_qca_oem.git/qsdk# ./scripts/feeds update -a
root@debian:~/wifi/qca-networking-2020-spf-11-1_qca_oem.git/qsdk# ./scripts/feeds install -a -f

Copy the base configuration to use for the build

Enterprise32-bit

root@debian:~/wifi/qca-networking-2020-spf-11-1_qca_oem.git/qsdk# cp qca/configs/qsdk/ipq_enterprise.config .config
root@debian:~/wifi/qca-networking-2020-spf-11-1_qca_oem.git/qsdk# sed -i "s/TARGET_ipq_ipq806x/TARGET_ipq_ipq60xx/g" .config

Regenerate a complete configuration file and start the build:

root@debian:~/wifi/qca-networking-2020-spf-11-1_qca_oem.git/qsdk# make defconfig
root@debian:~/wifi/qca-networking-2020-spf-11-1_qca_oem.git/qsdk# sed -i -e "/CONFIG_PACKAGE_qca-wifi-fw-hw5-10.4-asic/d" .config
root@debian:~/wifi/qca-networking-2020-spf-11-1_qca_oem.git/qsdk# make V=s

根据PC的配置，make V=s 编译过程持续1 ~ 4 小时，耐心等待 ~ 可以先干点别的事情了
